## WordMate 应用完整功能页面规格说明书
一、 核心学习流页面
1. 首页（消息列表）

布局：仿微信聊天列表。

列表项：

头像：单词首字母+背景色，右上角有红色数字角标（待复习消息数）。

主文本：单词名（如 Ambiguous）。

副文本：最后一条消息预览（如：“你能分清我和‘vague’吗？”）。

时间：最后互动时间（如“10:30”）。

按钮/交互：

点击列表项：进入该单词的 “启动与语音验证页”。

右滑列表项：露出绿色对勾图标和“记住了”文字，松开即执行 “打勾” 操作（单词从首页暂时消失，进入下一复习周期）。

左滑列表项：露出红色叉号图标和“忘记了”文字，松开即执行 “打叉” 操作（单词立即发送一条强化复习消息，且在首页排序提前）。

顶部搜索框（可选）：快速筛选单词好友。

2. 启动与语音验证页

触发：从首页或联系人页点击任一单词进入。

按钮/交互：

顶部喇叭按钮：点击重播单词标准发音。

中央大圆形录音按钮：长按开始录音跟读，松开结束。录音时按钮变红并有动画。

系统逻辑：录音后自动比对，相似度≥70%则弹出 “自觉测试弹窗”；否则显示红色提示文字，并可重试。

3. 自觉测试弹窗

触发：语音验证通过后自动弹出。

按钮：

【不，再教我一次】（红色边框）：关闭弹窗，跳转至 “单词详情介绍页”。

【是的，我掌握了】（绿色填充）：关闭弹窗，直接进入 “单词私聊主页面”。

4. 单词详情介绍页

进入方式：自觉测试弹窗选择“不”或从私聊页菜单点击“查看单词档案”。

内容：单词、音标、高频标签、详细释义（常考义高亮）、词根词缀动态图、权威例句。

按钮/交互：

喇叭按钮：播放发音。

例句中的单词：可点击，弹出该单词的简明释义浮层。

底部【进入聊天】按钮：点击进入 “单词私聊主页面”。

5. 单词私聊主页面

顶部导航栏：

返回箭头：返回上一页。

单词与掌握度：显示单词和掌握度百分比。

更多菜单（三点图标）：点击弹出菜单：

加入/移出白名单：切换单词的白名单状态。
查看单词档案：跳转到 “单词详情介绍页”。
记忆挑战：发起针对此单词的限时测试。
清除聊天记录：仅清空当前对话UI。
对话区域：

AI气泡内的任意单词：可点击，弹出该单词的快捷释义浮层。

底部输入栏：

【+】按钮：点击展开 “AI功能扩展面板”。

文本输入框：自由输入问题。

【发送】按钮：发送文本消息。

6. AI功能扩展面板

触发：私聊页点击输入栏左侧【+】。

布局：2x2网格的四个功能按钮。

【记忆技巧】（灯泡图标）：点击后自动发送一条请求记忆方法的指令。

【造句】（笔图标）：点击后弹出次级面板，选择或输入话题（如“旅行”），发送造句指令。

【比较】（天平图标）：点击后弹出输入框，输入近义词（如“vague”），发送比较指令。

【手写】（手写笔图标）：点击直接进入 “全屏手写板”。

7. 全屏手写板

触发：从AI扩展面板或复习测试中进入。

顶部栏：

【取消】：放弃并返回。

【完成/发送】：将手写内容生成图片，发送到聊天或提交为答案。

底部工具栏：

笔刷/颜色选择：黑/红/蓝，三种粗细。

【橡皮擦】：切换为擦除模式。

【撤销】/【重做】。

【清空画布】。

8. 白名单页面

布局：上部为统计概览，下部为单词网格列表。

按钮/交互：

【开始今日特训】：系统智能组合3-5个白名单单词生成混合题型挑战。

单词卡片：点击进入该单词的 “强化训练模式”；长按弹出菜单（移出白名单、查看详情）。

9. 白名单单词强化训练模式

布局：线性通关流程，依次为：

听力速记：听句子选关键词。
完形填空：在短文中选择正确单词填空。
主动输出：根据主题使用该单词造句，AI评估。
按钮：每个模块均有 【提交】或【下一关】 按钮。

二、 管理与发现页面
10. 联系人页

布局：仿微信通讯录，按字母A-Z排序所有“单词好友”。

按钮：

右上角【+】：点击进入 “搜索与添加新词页”。

列表项：点击任一单词，进入 “启动与语音验证页”。

11. 搜索与添加新词页

布局：顶部为搜索框。

交互：输入单词后，系统验证并显示基本信息。

按钮：出现 【加为好友】 按钮，点击后正式添加该单词。

三、 “我的”个人中心及设置流页面
12. “我的”主页

顶部个人信息区：

头像：可点击，弹出菜单：[拍照]、[从相册选择]、[查看大头像]。

昵称：可点击，进入编辑资料浮层。

数据卡片区（所有数字均可点击，跳转至 “学习数据统计页”）：

核心指标：连续天数、掌握单词、正确率。

今日学习：新词、复习、时长（点击跳转至首页对应过滤视图）。

图表区：

掌握度分布图：点击任一柱状条，跳转至 “我的词库管理页” 并自动筛选。

功能入口列表：

【我的词库】 -> 进入 “我的词库管理页”。

【数据看板】 -> 进入 “学习数据统计页”。

【会员订阅】 -> 进入 “会员与订阅中心”。

【设置】 -> 进入 “设置页”。

【帮助与反馈】 -> 进入 “帮助与反馈页”。

13. 学习数据统计页

顶部：< 返回，分享按钮（生成报告长图）。

标签：本周、本月、全部（切换时间范围）。

内容：记忆曲线、学习热力图、掌握度趋势图等图表。旗舰版用户有专属的AI学习建议区块。

14. 我的词库管理页

顶部：搜索框，筛选/排序选项。

列表：单词列表，左侧带复选框。

底部操作栏：

常态：全选、加入/移出白名单、删除、导出。

编辑模式：顶部显示已选X个单词和完成按钮。

15. 会员与订阅中心

内容：当前状态、权益对比表格（基础/高级/旗舰）。

订阅选项：高级版、旗舰版的价格卡片，点击选中。

底部按钮：

【恢复购买】：小字按钮。

【确认订阅】：大按钮，触发应用内购买流程。

16. 设置页

列表项（每项右侧有 > 箭头）：

通知管理：进入子页，开关控制。
发音设置：进入子页，英音/美音单选。
AI偏好：进入子页，设置对话风格与长度。
数据与存储：进入子页，显示缓存大小，含清理缓存按钮。
账号与安全：进入子页，含绑定手机、修改密码、注销账号（红字）。
关于我们：进入子页，显示版本号、协议链接。
退出登录：居中红色按钮，点击弹出确认框。
17. 帮助与反馈页

内容：常见问题（FAQ）折叠列表。

反馈表单：问题反馈文本框、上传截图按钮、联系方式输入框。

按钮：底部 【提交】 按钮。

四、 系统与状态页面
18. 登录/注册页

输入框：手机号、验证码。

按钮：获取验证码、登录/注册、微信登录、Apple登录。

19. 启动/闪屏页：显示Logo和Slogan，无交互。

20. 空状态页

首页空状态：引导图文 + 【去添加第一个单词好友】 按钮。

白名单空状态：引导图文，解释白名单作用。

21. 加载/等待状态：页面骨架屏或“思考中…”微文案。

22. 网络错误页：显示“网络开小差了” + 【重试】 按钮。

五、 关键交互流程总结
复习主循环：首页 -> 点击单词 -> 语音验证 -> 自觉测试 -> 私聊页互动 -> 返回首页滑卡打勾。

添加新词：联系人页 -> 点击【+】 -> 搜索与添加页 -> 输入并点击【加为好友】 -> 新词出现在首页并发送第一条消息。

管理难词：在私聊页菜单 -> 点击加入白名单 -> 前往白名单页 -> 点击卡片进入强化训练。

查看数据：我的主页 -> 点击数据图表或【数据看板】 -> 学习数据统计页。
